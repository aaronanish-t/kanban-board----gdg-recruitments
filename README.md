# 📋 Professional Kanban Board

A sleek, high-performance Kanban Task Management application built with **React** and **Vite**. This project was designed with a focus on professional UI/UX, state persistence, and interactive user experience.

## 🚀 Live Demo
https://kanban-board-gdg-recruitments.vercel.app/

---

## ✨ Features

* **Drag & Drop Interface:** Seamlessly move tasks between `Todo`, `In Progress`, and `Done` columns using the native HTML5 Drag and Drop API.
* **🌓 Adaptive Dark Mode:** A fully integrated dark/light theme toggle that persists across sessions.
* **💾 Local Persistence:** Your tasks and theme preferences are saved to `localStorage`, ensuring data stays safe after page refreshes.
* **📱 Responsive Design:** Optimized for both desktop and mobile viewing with a flexible CSS Grid and Flexbox layout.
* **⚡ Built with Vite:** Leveraging the speed of Vite for ultra-fast development and optimized production builds.
* **🧹 QoL Features:** * Add tasks by pressing **Enter**.
    * Clear the input field with **Esc**.
    * "Clear All" button to reset the board.
    * Dynamic task counters for each column.

---

## 🛠️ Tech Stack

* **Frontend:** React (Hooks: `useState`, `useEffect`)
* **Styling:** CSS3 (Variables, Flexbox, Grid, Glassmorphism)
* **Build Tool:** Vite
* **Deployment:** Vercel
* **Version Control:** Git & GitHub
